# Gemini Context: vms-mini-app

## Project Overview
This is a Next.js 14 application named "vms-mini-app". It uses the App Router architecture.

## Tech Stack
- **Framework**: Next.js 14.2.5
- **UI Library**: React 18.3.1
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Next.js compiler

## Key Conventions
- **Routing**: Uses the Next.js App Router (`app/` directory).
- **Styling**: Utility-first CSS using Tailwind CSS.
- **Component Style**: Functional components with TypeScript interfaces for props.
- **File Structure**: Colocation of related files is encouraged where appropriate.

## Commands
- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run start`: Start production server
- `npm run lint`: Run linter
